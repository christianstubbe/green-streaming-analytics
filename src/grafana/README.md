# Grafana Dashboard

## Local Installation

### Prerequisities

- Docker

### Setup

```docker compose up -d```

Open your web browser and go to http://localhost:3000. On the signin page, enter admin for username and password. Click Sign in. If successful, you will see a prompt to change the password. Click Skip.
